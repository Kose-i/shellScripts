# shellScripts
shellScriptsは様々なソフトウェアの設定をシェルスクリプトにしたものです．

**※あなたの設定を上書きする場合がありますのでご注意ください．**

## gitSetting
gitに関する設定を行います．
setAll.sh を実行すると同じ階層のすべてのシェルスクリプトが実行されますのでご利用ください．

## aptRunner
apt-getの以下を行います

- update
- upgrade
- autoremove
- clean
